# java-racingcar-precourse
## 기능 리스트
### 입력
#### 1. 자동차 이름 입력 받기
- 쉼표로 구분
- 5자 이하
- 입력 값 잘못 받았을 때 예외처리 `IllegalArgumentException`

#### 2. 시도할 횟수 입력 받기

### 출력
#### 1. 전진하기
- 전진조건 : 0~9 사이 무작위 값이 4 이상인 경우 전진
- 자동차 이름과 함께 전진여부를 표시함.
    - ex) `pobi : --`
- 전진한 내용을 실행결과로 보여줌.

#### 2. 경기 완료, 우승자 출력하기
- 우승자는 한명이상임
- 우승자가 여러명일 경우 쉼표로 구분함
