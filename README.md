# java-racingcar-precourse

# **📋 기능 목록**

### InputView

- [x]  자동차 이름 입력
   - [x]  예외처리 - 이름 유효성 검증 (5자 이하)
   - [x]  예외처리 - 공백
- [x]  시도 횟수 입력
   - [x]  예외처리 - 시도 횟수 유효성 검증 (시도 횟수가 자연수가 아닌 경우)

### CarFactory

- [x]  자동차 객체 생성 - 기본 랜덤값 필드 0으로 주고 시작

### Car

- [x]  자동차별 랜덤 값 발생

### Race

- [x]  n번 경주 진행
- [x]  경주 결과 판단

### OutputView

- [x]  경주를 반복할 때마다 진행상황 출력
- [x]  우승자 출력
