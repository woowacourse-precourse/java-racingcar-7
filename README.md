# 자동차 경주

<br>

### 구현 기능 목록
1. 입력
   - 자동차 이름
   - 이동 횟수
   - [예외 처리] 자동차 이름이 5자를 초과한 경우
   - [예외 처리] 다수의 자동차 이름 입력 시 쉼표 이외의 구분자를 사용한 경우
   - [예외 처리] 이동 횟수가 숫자가 아닌 경우

2. 문자열 처리
    - 다수의 자동차 이름을 쉼표로 분리
    - 공동 우승자를 쉼표로 구분
     
3. 이동 처리
    - 0~9 사이의 무작위 값이 4 이상이면 전진, 4 미만이면 정지
    - 모든 이동이 끝난 후 전진 횟수가 가장 많은 자동차 구하기 
    
4. 출력
    - 전진 결과
    - 우승자 안내 문구
