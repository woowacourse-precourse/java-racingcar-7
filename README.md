# 자동차 경주

---

## :white_check_mark: 기능 목록
- 자동차 이름 입력
  - (,) 쉼표로 구분
  - 이름이 5자를 초과하는 경우 예외 처리
  - 중복된 이름이 존재하는 경우 예외 처리
  - 공백 문자만 입력된 경우 예외 처리
- 이동 횟수 입력
  - 숫자가 아닌 경우 예외 처리
- 실행 결과 출력
- 최종 결과 출력
- 자동차 전진 기능
- 전진 조건 판단 기능
- 랜덤 값 추출 기능
