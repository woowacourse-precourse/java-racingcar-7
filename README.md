# java-racingcar-precourse

### 자동차 경주 요구사항

#### Cars

+ [x] 자동차 이름을 분리하여 자동차 객체 생성
+ [x] 최대 이동 거리를 구할 수 있다.
+ [x] 우승자를 구할 수 있다.

#### Car

+ [x] 경주 자동차 이름이 5글자를 초과할 수 없으며 빈칸일 수 없다.
+ [x] 숫자가 3이하인 경우 전진할 수 없다.
+ [x] 숫자가 4이상인 경우 전진할 수 있다.

#### Distance

+ [x] 거리가 증가가능하다.
+ [x] 최대 이동 거리와 같은지 구할 수 있다.
+ [x] 최대 이동 거리를 구할 수 있다.

#### TryCount

+ [x] 시도횟수가 숫자가 아니거나 0이하일 수 없다.
+ [x] 시도횟수 차감
+ [x] 시도횟수 남았는지 판별
