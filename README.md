### 기능 목록
- 입력
    - [] 자동차 이름 (쉼표로 구분, 5자 이하)
    - [] 시도할 횟수
- 게임 실행
    - [] 자동차 이름, 입력 횟수 주입
    - [] 시도할 횟수만큼 각 자동차 전진
    - [] 자동차 별 전진 실행 결과 계산
- 출력
  - [] 자동차 별 전진 실행 결과 조회
  - [] 전진 실행 결과가 큰 순서대로 정렬
  - [] 정렬된 결과를 통해 최종 우승자 계산
  - [] 최종 우승자 문구 출력
- 전진
  - [] 0~9 랜덤값 생성 결과가 4이상인 경우 전진
