# java-racingcar-precourse

# **📋 기능 목록**

### InputView

- [ ]  자동차 이름 입력
   - [ ]  예외처리 - 이름 유효성 검증 (5자 이하)
   - [ ]  예외처리 - 공백
- [ ]  시도 횟수 입력
   - [ ]  예외처리 - 시도 횟수 유효성 검증 (시도 횟수가 자연수가 아닌 경우)

### CarFactory

- [ ]  자동차 객체 생성

### Car

- [ ]  자동차별 랜덤 값 발생

### Race

- [ ]  경주 진행 
- [ ]  경주 결과 판단

### OutputView

- [ ]  경주를 반복할 때마다 진행상황 출력
- [ ]  우승자 출력
