- [ ] 자동차 이름(플레이어 이름)

- [ ] 정지
### 플레이어
- [ ] 플레이어 이름
- [ ] 승리 횟수
### 우승자
- [ ] 우승자 이름

- 자동차 입력

  - [ ] 인트범위를 넘는 수
